# 🎭 역할 기반 AI 챗봇 (Role-based Streamlit Chatbot)

이 프로젝트는 Streamlit과 OpenAI API를 사용하여 특정 페르소나(역할)를 가진 AI와 대화할 수 있는 간단한 챗봇 애플리케이션입니다.

## 🛠️ 사용 기술

* **프론트엔드/UI:** Streamlit (Python)
* **백엔드/AI:** OpenAI API (ChatGPT)

## 🚀 로컬 실행 방법

1.  **환경 설정:** Python 3.x 환경이 필요합니다.
2.  **라이브러리 설치:**
    ```bash
    pip install -r requirements.txt
    ```
3.  **API 키 설정:**
    * OpenAI API 키를 발급받으세요.
    * 프로젝트 폴더 내에 `.streamlit` 폴더를 만들고, 그 안에 `secrets.toml` 파일을 생성하여 키를 저장합니다.
    ```toml
    # .streamlit/secrets.toml
    [openai]
    api_key = "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"
    ```
4.  **애플리케이션 실행:**
    ```bash
    streamlit run app.py
    ```

## ☁️ Streamlit Cloud 배포

이 프로젝트는 GitHub에 푸시된 후, Streamlit Cloud를 통해 쉽게 배포될 수 있습니다. 배포 시 **"Secrets"** 설정에 OpenAI API 키를 입력해야 합니다.
